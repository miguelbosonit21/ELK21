docker-compose ps                          -> Ver contenedores levantados
docker-compose up -d                       -> para ver los logs por pantalla
docker-compose up                          -> Levantar todo el stack (docker-compose.yml)
docker-compose down                        -> Tirar todo el stack (docker-compose.yml)
docker-compose start elasticsearch         -> Levantar contenedor concreto
docker-compose stop elasticsearch          -> Tirar contenedor concreto
